# 乐挑促销Android项目架构：

+ 网络及异步框架：okhttp3 + retrofit2 + rxjava2
+ RPC数据解析：gson
+ 图片加载：Glide + 头像circleimageview
+ 列表加载：BaseRecyclerViewAdapterHelper + easyrecyclerview
+ 表单验证：saripaar
+ facebook的网络监控：stetho
+ 日志：logger
+ 动态权限请求：rxpermissions
